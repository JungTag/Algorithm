입력은 동일하게 3(n), 1, 2, 3

코드1)
import sys
n = int(input())
input_list = [sys.stdin.readline().split() for i in range(n)]
print(input_list)
출력:[['1'],['2'],['3']]

코드2)
import sys
n = int(input())
input_list = [sys.stdin.readline().strip() for i in range(n)]
print(input_list)
출력:['1','2,'3']

#중요!: split은 쪼개서 리스트화 시킨다. 따라서 한 줄에 여러개의 입력이 있을 때 사용하자!
